This is an interactive 3D OpenGL visualization of the Lorenz Attractor.


Key bindings
  w/W    Increase s value
  s/S    Decrease s value
  
  a/A    Increase b value
  d/D    Decrease b value
  
  q/Q    Increase r value
  e/E    Decrease r value
  
  x/X    Reset s/b/r values
  
  arrows Change view angle
  0      Reset view angle
  
  ESC    Exit
